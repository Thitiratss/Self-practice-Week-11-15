หนูได้เรียนรู้การเขียนฟังก์ชั่น showErrorDialog โดยเอาจากที่อาจารย์สอนเรื่อง fetch ใน ค่ะที่มีพวก case แล้ว break ต่าง ๆ หนูนำมาปรับใช้เข้ากับโปรเจ็คของหนู ข้อความ dialog สามารถแสดงได้ตามเงื่อนไขของ res.status ใน code มี onOk, onCancel, onKeep ซึ่งเป็นฟังก์ชันที่ส่งเข้ามาจากภายนอก เพื่อปรับใช้ dialog ตามแบบที่ต้องการ สามารถเรียกใช้ได้เลยว่า เลือกให้มีปุ่ม OK หรือไม่ การกำหนดให้ปิดด้วยปุ่ม ESC ได้หรือไม่ รวมถึงการสร้าง dialog แบบ confirm ที่มีทั้งปุ่ม Cancel และ Keep ช่วยให้สามารถปรับการใช้งานได้หลากหลาย นอกจากนี้ยังสามารถกำหนดข้อความ message เองได้ ทำให้หนูเข้าใจการ handing error มากขึ้นและสามารถนำไปใช้ได้หลายส่วนในโปรเจ็คด้วยค่ะ
